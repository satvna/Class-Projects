------------------------------------------------------------------------------------

This program takes the headlines from the business, entertainment, and style 
pages of CNN. They are then output to a csv table.

------------------------------------------------------------------------------------

Requirements: 
	- Python 3
	- Beautiful Soup
	- Requests
	- html5lib

How to use: 

	1) Install the necessary libraries by running these commands:
		- pip install requests
		- pip install html5lib
		- pip install bs4
	2) Run "python webscraper.py" in the terminal.
	3) Done! You can find the output in the "headlines.csv" file.
